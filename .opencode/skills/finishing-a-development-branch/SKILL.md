# Finishing a Development Branch

## Overview

Close work intentionally with evidence-based readiness checks and explicit next-step options.

## Closeout checklist

- Scope completed and documented
- Mode-required verification completed
- Review findings resolved or acknowledged
- Risks and follow-ups captured

## Decision options

1. Continue work on branch
2. Open PR
3. Merge locally
4. Discard branch changes (explicit user request only)

## Output

- readiness status
- supporting evidence
- recommended path with rationale
